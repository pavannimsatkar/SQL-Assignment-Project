# SQL Assignment Project 
## Retail Store Analytics System 

### 🎯Objective 

You are working as a Data Analyst in a retail company. The management wants to analyze 
customer purchases, product performance, sales trends, payment methods, and city-wise 
business performance using SQL. 

### 📖Overview

This project is a SQL-based data analysis assignment focused on a Retail Store Analytics System. The objective is to simulate the role of a Data Analyst in a retail company and use SQL queries to extract meaningful business insights from sales and customer data.

The project demonstrates the use of SQL concepts such as filtering, aggregation, grouping, sorting, and advanced analytical queries to answer real-world business questions.
